My Projects

Check out some of the cool projects that I have worked on using a variety of tech stacks.

MyLeague

This project is based around the popular video game League of Legends. It allows the user to look at information about the various champions, items, runes and summoner spells in the game. It also allows users to perform CRUD operations on sets, where a set may contain a champion, and items/runes/summoner spells associated with those champions.

Find My Paradise

This is a project that I worked on as part of a 12 person team, which was tasked to create an alternate version of an Airbnb listing page. It shows all of the details of a listing and allows a user to create a reservation. My primary task as a contributor to this project was to implement the amenities component.

Nile

This is a other team project that I worked on with the same team as Find My Paradise. This project was a replica of Amazon, which has a home page, individual product listing pages, and various other functions such as adding to cart and checking out. My primary task as a contributor was to implement the "Customer Also Bought" scrolling list component.
